Primeiro Site Hospedado no GitHub

Passo a passo para Hospedagem no GitHub

Passo 01: Devemos criar um repositório para nosso projeto
Passo 02: Publicar o repositório no GitHub.com
Passo 03: Agora abrir no VsCode
Passo 04: Criar um arquivo index.html
Passo 05: Commitar no githubdesktop
Passo 06: Realizar o push
Passo 07: Agora no GitHub, dentro do seu repositório, clicar em settings > pages > source no select adicionar a option main e salvar
